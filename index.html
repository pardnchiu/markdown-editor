    # Markdown 簡易編輯器
    > 為 [JOBALL找專家](https://joball.tw "") 的需求所開發的功能
    用於提供給專家 (SOHO族 / 接案者) 們撰寫各自領域相關內容的編輯器
    md語法未完善適配，會持續更新補強
